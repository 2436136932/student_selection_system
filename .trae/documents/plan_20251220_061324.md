## 问题分析
所有页面滚轮失效的根本原因是全局样式设置了多处`overflow: hidden`：
1. `html, body { overflow: hidden; }` - 整个页面无法滚动
2. `.app-container { overflow: hidden; }` - 应用容器无法滚动
3. `.el-container { overflow: hidden; }` - Element Plus容器无法滚动
4. `.el-main { overflow: hidden; }` - 主内容区域无法滚动

这些全局样式导致所有页面的滚动都被阻止了。

## 解决方案
1. **修改App.vue的全局样式**：
   - 移除`html, body`的`overflow: hidden`
   - 移除`.app-container`的`overflow: hidden`
   - 移除`.el-container`的`overflow: hidden`
   - 修改`.el-main`的样式，添加`overflow: auto`

2. **为聊天中心添加特定样式**：
   - 确保聊天中心页面的布局仍然正常
   - 只在聊天中心的特定区域使用`overflow: hidden`
   - 确保聊天消息区域可以正常滚动

## 预期效果
修改后，所有页面的滚轮都将恢复正常工作，而聊天中心的布局和滚动功能也将保持正常。
