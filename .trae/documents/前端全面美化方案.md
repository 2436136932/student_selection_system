# 前端美化实施计划

## 📋 概述
对整个前端系统进行全面美化，包括排版优化、卡片设计、表格美化、表单元素、加载状态、通知消息和聊天中心增强。

## 🎯 实施步骤

### 1. 全局样式优化
- **创建全局样式文件** `frontend/src/styles/global.css`
  - 引入现代字体（Inter、Noto Sans SC）
  - 定义统一的CSS变量（颜色、间距、字体大小）
  - 优化全局字体大小和行高
  - 增加标题和内容的间距

### 2. 卡片和容器美化
- **优化 App.vue 中的全局卡片样式**
  - 添加毛玻璃效果（backdrop-filter: blur）
  - 增加卡片圆角至12-16px
  - 优化阴影效果，使用多层阴影
  - 添加边框渐变

### 3. 表格美化
- **优化所有表格组件**（StudentAwardApplicationView.vue、StudentView.vue等）
  - 添加表格行hover效果（渐变背景）
  - 状态标签使用更醒目的颜色
  - 添加表格排序和筛选的视觉提示
  - 优化分页器样式（圆角、渐变按钮）

### 4. 表单元素优化
- **创建浮动标签组件** `frontend/src/components/FloatingLabelInput.vue`
- **优化所有表单页面**
  - 优化输入框focus状态（发光效果、边框渐变）
  - 添加浮动标签效果
  - 表单验证错误使用更友好的提示（带动画）
  - 文件上传区域添加拖拽样式

### 5. 加载状态
- **创建骨架屏组件** `frontend/src/components/SkeletonLoader.vue`
- **优化所有加载状态**
  - 添加骨架屏加载效果
  - 优化加载动画（旋转、脉冲）
  - 添加进度提示

### 6. 通知和消息
- **优化 App.vue 中的通知样式**
  - 优化通知样式（毛玻璃、渐变）
  - 添加通知动画（滑入、淡出）
  - 支持通知堆叠显示

### 7. 聊天中心增强
- **优化 ChatWindow.vue**
  - 优化聊天气泡样式（渐变、圆角、阴影）
  - 添加消息动画（滑入、弹跳）
  - 添加图片上传功能
  - 添加表情包选择器
  - 优化输入区域（工具栏、拖拽上传）

- **优化 ChatList.vue**
  - 优化用户列表项样式
  - 添加在线状态动画
  - 优化未读消息提示

### 8. 数据库字段更新
- **更新 ChatMessage 实体**
  - 添加 `messageType` 字段（text/image/emoji）
  - 添加 `imageUrl` 字段（存储图片URL）
  - 添加 `emoji` 字段（存储表情）

- **更新 ChatController**
  - 支持图片上传接口
  - 支持表情消息发送

## 📦 需要创建的文件
1. `frontend/src/styles/global.css` - 全局样式
2. `frontend/src/components/SkeletonLoader.vue` - 骨架屏组件
3. `frontend/src/components/FloatingLabelInput.vue` - 浮动标签输入框
4. `frontend/src/components/EmojiPicker.vue` - 表情选择器
5. `frontend/src/components/ImageUpload.vue` - 图片上传组件

## 🔧 需要修改的文件
1. `frontend/src/App.vue` - 全局样式、通知优化
2. `frontend/src/views/ChatView.vue` - 聊天中心容器
3. `frontend/src/components/ChatWindow.vue` - 聊天窗口
4. `frontend/src/components/ChatList.vue` - 聊天列表
5. `frontend/src/views/StudentAwardApplicationView.vue` - 表格美化
6. `frontend/src/views/StatisticsView.vue` - 卡片美化
7. `frontend/src/main.js` - 引入全局样式
8. `frontend/src/main/resources/application.yml` - 配置文件上传路径

## 🎨 设计要点
- 使用渐变色增强视觉效果
- 添加流畅的过渡动画
- 优化间距和排版
- 提升交互反馈
- 支持深色模式
