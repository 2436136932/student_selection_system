# 显示设置功能实现计划

## 一、当前状态分析

### 现有功能
1. **UI界面**：显示设置页面已创建，包含四个设置项
2. **数据绑定**：设置项已与Vue响应式数据绑定
3. **状态保存**：设置项已保存到localStorage
4. **主题功能**：颜色主题切换功能已实现

### 未实现功能
1. **紧凑模式**：未实现组件间距调整
2. **侧边栏折叠**：未实现默认折叠功能
3. **面包屑显示**：未实现显示/隐藏控制
4. **深色模式**：未实现深色/浅色主题切换

## 二、功能说明

### 1. 紧凑模式
- **功能**：减少组件间距，适合小屏设备
- **实现方式**：动态添加/移除CSS类，调整组件间距
- **影响范围**：所有页面的组件间距

### 2. 侧边栏折叠
- **功能**：默认折叠侧边栏
- **实现方式**：从localStorage读取设置，设置侧边栏初始状态
- **影响范围**：侧边栏的折叠状态

### 3. 面包屑显示
- **功能**：在页面顶部显示/隐藏面包屑导航
- **实现方式**：根据设置动态显示/隐藏面包屑组件
- **影响范围**：所有页面的面包屑导航

### 4. 深色模式
- **功能**：切换深色/浅色主题
- **实现方式**：动态添加/移除CSS类，调整系统颜色
- **影响范围**：整个系统的颜色主题

## 三、实现步骤

### 1. 紧凑模式实现
- 修改App.vue，添加紧凑模式CSS类
- 在main-content区域添加动态CSS类绑定
- 从localStorage读取设置，初始化为响应式数据

### 2. 侧边栏折叠实现

- 初始化时设置侧边栏折叠状态
- 在显示设置页面添加同步逻辑

### 3. 面包屑显示实现
- 修改App.vue，在面包屑组件上添加v-if条件
- 从localStorage读取面包屑显示设置
- 在显示设置页面添加同步逻辑

### 4. 深色模式实现
- 修改App.vue，添加深色模式CSS类
- 在根元素上添加动态CSS类绑定
- 从localStorage读取深色模式设置
- 实现深色/浅色主题切换逻辑
- 在显示设置页面添加同步逻辑

### 5. 系统启动时初始化设置
- 修改App.vue，在组件挂载时初始化所有显示设置
- 确保设置项从localStorage正确读取
- 确保设置项正确应用到UI

## 四、技术实现

### 1. CSS实现
- 使用CSS变量和CSS类实现主题切换
- 为紧凑模式和深色模式创建专门的CSS类
- 使用!important确保样式优先级

### 2. JavaScript实现
- 使用Vue 3的Composition API
- 使用localStorage进行状态持久化
- 使用响应式数据进行状态管理
- 使用watch监听设置变化，实时更新UI

### 3. 组件交互
- 在显示设置页面修改设置时，实时更新UI
- 确保设置保存后，下次登录时自动应用
- 提供重置默认设置功能

## 五、预期效果

1. **紧凑模式**：开启后，组件间距变小，适合小屏设备
2. **侧边栏折叠**：开启后，下次登录时侧边栏默认折叠
3. **面包屑显示**：关闭后，页面顶部不再显示面包屑导航
4. **深色模式**：开启后，系统切换为深色主题，减轻眼部疲劳
5. **实时生效**：修改设置后，效果实时显示，不需要刷新页面
6. **持久化保存**：设置保存到localStorage，下次登录时自动加载
