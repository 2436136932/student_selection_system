# 修复奖项申请趋势图表数据显示问题

## 问题分析

根据提供的信息，数据库中存在`student_award_records`记录，但是可视化图表没有显示数据。通过检查代码，我发现：

1. API端点`/api/statistics/application-trend`现在调用的是`studentAwardRecordService.getAwardTrend()`方法
2. 该方法使用`DATE_FORMAT(award_time, '%Y-%m')`按月份统计获奖记录数量
3. 但是API返回的是最近6个月的默认数据（值都为0），说明查询没有返回任何结果

## 解决方案

修改`StudentAwardRecordServiceImpl`类中的`getAwardTrend()`方法，使其：

1. 使用`COALESCE(award_time, created_at)`来处理可能为空的`award_time`字段
2. 保持原有的月份分组和排序逻辑
3. 添加日志记录，以便调试

## 实现步骤

1. 修改`StudentAwardRecordServiceImpl`类中的`getAwardTrend()`方法，将`DATE_FORMAT(award_time, '%Y-%m')`替换为`DATE_FORMAT(COALESCE(award_time, created_at), '%Y-%m')`
2. 添加日志记录，打印查询结果
3. 重新编译和启动服务
4. 测试API端点，确保返回正确的数据

## 代码修改

```java
@Override
public Map<String, Long> getAwardTrend() {
    Map<String, Long> trend = new java.util.LinkedHashMap<>();
    
    try {
        // 使用MyBatis Plus的SQL查询直接按月份统计获奖记录数量
        List<java.util.Map<String, Object>> monthCounts = studentAwardRecordMapper.selectMaps(
                new com.baomidou.mybatisplus.core.conditions.query.QueryWrapper<StudentAwardRecord>()
                        .select("DATE_FORMAT(COALESCE(award_time, created_at), '%Y-%m') as month, count(*) as count")
                        .groupBy("DATE_FORMAT(COALESCE(award_time, created_at), '%Y-%m')")
                        .orderByAsc("DATE_FORMAT(COALESCE(award_time, created_at), '%Y-%m')")
        );
        
        // 打印查询结果，用于调试
        System.out.println("获奖记录趋势查询结果: " + monthCounts);
        
        // 构建月份计数映射
        for (java.util.Map<String, Object> monthCount : monthCounts) {
            String month = monthCount.get("month").toString();
            Long count = Long.valueOf(monthCount.get("count").toString());
            trend.put(month, count);
        }
    } catch (Exception e) {
        System.err.println("获取奖项记录趋势失败: " + e.getMessage());
        e.printStackTrace();
    }
    
    // 如果没有数据，添加最近6个月的默认数据，值为0
    if (trend.isEmpty()) {
        java.util.Calendar cal = java.util.Calendar.getInstance();
        for (int i = 5; i >= 0; i--) {
            java.util.Calendar tempCal = java.util.Calendar.getInstance();
            tempCal.add(java.util.Calendar.MONTH, -i);
            java.text.SimpleDateFormat sdf = new java.text.SimpleDateFormat("yyyy-MM");
            String month = sdf.format(tempCal.getTime());
            trend.put(month, 0L);
        }
    }
    
    return trend;
}
```
