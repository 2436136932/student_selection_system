## 实现需求
在系统上方右侧的身份左边添加"系统当前在线人数："文字，并显示真实的在线人数。

## 实现方案
1. **添加在线人数数据**：在App.vue中添加响应式数据`onlineUserCount`用于存储在线人数
2. **添加获取在线人数方法**：实现`getOnlineUserCount`方法，调用后端`/api/chats/online-users`接口获取在线用户列表，计算人数
3. **定期更新在线人数**：在组件挂载时获取一次在线人数，并设置定时器每30秒更新一次
4. **修改UI显示**：在header-info中添加显示在线人数的DOM元素，位置在角色标签左边

## 实现代码
1. **在App.vue的script中添加**：
   ```javascript
   // 在线人数
   const onlineUserCount = ref(0)
   
   // 获取在线人数
   const getOnlineUserCount = async () => {
     try {
       const response = await axios.get('/api/chats/online-users', {
         headers: {
           'Authorization': `Bearer ${localStorage.getItem('token')}`
         }
       })
       if (response.data) {
         onlineUserCount.value = response.data.length || 0
       }
     } catch (error) {
       console.error('获取在线人数失败:', error)
     }
   }
   
   // 在onMounted中添加
   onMounted(() => {
     // 现有代码...
     
     // 获取在线人数
     getOnlineUserCount()
     // 每30秒更新一次在线人数
     setInterval(getOnlineUserCount, 30000)
   })
   ```

2. **在App.vue的template中修改**：
   ```vue
   <div class="header-info">
     <span class="online-count">系统当前在线人数：{{ onlineUserCount }}</span>
     <span class="role-tag" :class="getRoleClass()">
       {{ getRoleName() }}
     </span>
     <!-- 现有用户信息代码... -->
   </div>
   ```

3. **在App.vue的style中添加**：
   ```css
   .online-count {
     font-size: 14px;
     color: #606266;
     margin-right: 15px;
   }
   ```

## 预期效果
- 在系统顶部右侧，角色标签左边显示"系统当前在线人数：X"
- 在线人数会定期更新，显示真实的在线用户数量
- 样式与现有UI保持一致，不影响其他功能

## 技术要点
- 使用axios调用后端接口获取在线用户列表
- 使用setInterval定期更新数据
- 响应式数据实时更新UI
- 保持与现有样式的一致性
