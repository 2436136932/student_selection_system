# 实现登录界面忘记密码功能的详细计划

## 一、功能概述
通过邮箱发送验证码，让用户能够重置密码，包括以下步骤：
1. 用户输入邮箱，点击发送验证码
2. 系统向用户邮箱发送验证码
3. 用户输入验证码，点击下一步
4. 用户输入新密码，完成密码重置

## 二、数据库设计
1. **添加验证码表**：创建`verification_codes`表用于存储验证码信息

## 三、后端实现

### 1. 创建实体类
- `VerificationCode.java`：验证码实体类，包含id、email、code、expireTime、createdAt等字段

### 2. 创建Mapper/Repository
- `VerificationCodeRepository.java`：验证码数据访问接口，包含保存、查询、删除等方法

### 3. 创建Service
- `VerificationCodeService.java`：验证码服务接口，包含生成、发送、验证、删除验证码等方法
- `VerificationCodeServiceImpl.java`：验证码服务实现

### 4. 创建Controller
- 在`AuthController`中添加以下API：
  - `sendVerificationCode`：发送验证码
  - `verifyCode`：验证验证码
  - `resetPassword`：重置密码

### 5. 配置邮箱服务
- 在`application.yml`中添加邮箱配置

## 四、前端实现

### 1. 修改LoginView.vue
- 在登录表单中添加"忘记密码"链接
- 创建忘记密码的模态框或页面，包含：
  - 邮箱输入表单
  - 验证码输入表单
  - 新密码输入表单
- 实现表单验证和提交逻辑
- 添加倒计时功能（验证码有效期）

### 2. 添加API调用
- 实现发送验证码的API调用
- 实现验证验证码的API调用
- 实现重置密码的API调用

## 五、实现步骤

1. **创建数据库表**：在`database_init.sql`中添加`verification_codes`表的创建语句
2. **实现后端验证码服务**：创建实体类、Mapper、Service和Controller
3. **配置邮箱服务**：在`application.yml`中添加邮箱配置
4. **实现前端界面**：修改LoginView.vue，添加忘记密码功能
5. **测试功能**：验证忘记密码流程是否正常工作

## 六、技术要点

- 使用Spring Boot Mail发送邮件
- 验证码有效期设置为10分钟
- 每次发送新验证码时，删除旧的验证码
- 使用UUID生成随机验证码
- 密码加密使用现有的PasswordEncoder
- 前端使用Element Plus的表单和模态框组件
- 添加适当的错误处理和用户提示

## 七、安全考虑

- 限制同一邮箱的验证码发送频率
- 验证码存储使用数据库，避免内存泄漏
- 验证码过期后自动删除
- 密码重置成功后删除验证码
- 使用HTTPS确保数据传输安全
