# 修复右上角小头像显示问题

## 问题分析

根据代码分析，右上角小头像不显示的原因是：

1. **组件属性使用错误**：在App.vue中，右上角头像使用了`:icon`属性来显示头像图片，而正确的属性应该是`:src`
2. **头像URL处理问题**：没有将相对路径转换为完整URL，可能导致跨端口访问失败
3. **缺少错误处理**：没有处理图片加载失败的情况
4. **用户信息同步问题**：个人中心更新头像后，右上角头像可能没有同步更新

## 解决方案

### 核心思路

1. **修复el-avatar组件属性**：将`:icon`属性改为`:src`属性，用于显示头像图片
2. **添加完整URL转换**：确保头像URL使用完整的URL格式
3. **添加默认图标显示**：当头像URL为空时，显示默认的用户图标
4. **添加错误处理**：处理图片加载失败的情况
5. **确保用户信息同步**：确保个人中心更新头像后，右上角头像同步更新

### 具体修改步骤

1. **修改App.vue中的右上角头像组件**：
   - 将`:icon`属性改为`:src`属性
   - 添加`:error`事件处理
   - 添加默认图标显示逻辑
   - 添加头像URL转换逻辑

2. **修改用户信息处理逻辑**：
   - 确保从localStorage获取的用户信息包含完整的头像URL
   - 确保个人中心更新头像后，localStorage中的用户信息也同步更新

### 具体修改代码

#### 1. 修改App.vue中的右上角头像组件

```vue
<el-dropdown trigger="click" @command="handleDropdownCommand">
  <div class="user-info">
    <el-avatar 
      size="small" 
      :src="getFullAvatarUrl(userInfo.avatar)"
      @error="handleAvatarError"
    >
      <el-icon v-if="!userInfo.avatar"><UserFilled /></el-icon>
    </el-avatar>
    <span class="user-name">{{ userInfo.name || userInfo.username }}</span>
    <el-icon class="el-icon--right"><arrow-down /></el-icon>
  </div>
  <!-- 下拉菜单内容 -->
</el-dropdown>
```

#### 2. 添加相关方法

```javascript
// 获取完整的头像URL
const getFullAvatarUrl = (avatar) => {
  if (!avatar) return null
  // 如果是相对路径，添加后端域名和端口
  if (avatar.startsWith('/')) {
    return `http://localhost:8080${avatar}`
  }
  return avatar
}

// 处理头像加载错误
const handleAvatarError = (error) => {
  console.error('右上角头像加载失败：', error)
}
```

#### 3. 确保用户信息同步

在ProfileView.vue的confirmAvatarUpload方法中，添加同步更新localStorage的逻辑：

```javascript
// 更新用户头像后，同步更新localStorage中的用户信息
const updateLocalStorage = (userInfo) => {
  const currentUserInfo = JSON.parse(localStorage.getItem('userInfo') || '{}')
  localStorage.setItem('userInfo', JSON.stringify({
    ...currentUserInfo,
    ...userInfo
  }))
}

// 在confirmAvatarUpload方法中添加
updateLocalStorage(userInfo.value)
```

### 预期效果

1. **右上角头像正确显示**：
   - 当用户有头像时，显示用户头像图片
   - 当用户没有头像时，显示默认的用户图标
   - 头像图片加载失败时，显示默认的用户图标

2. **头像URL正确处理**：
   - 相对路径被转换为完整URL，解决跨端口访问问题
   - 外部URL直接使用

3. **用户信息同步更新**：
   - 个人中心更新头像后，右上角头像同步更新
   - 刷新页面后，右上角头像仍然显示正确
