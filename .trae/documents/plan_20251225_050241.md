# 实现获奖记录功能计划

## 1. 需求分析

1. 增加一个新表用来存储获奖学生的记录信息
2. 在左边菜单栏新增获奖记录功能，并将奖项申请和获奖记录合并为奖项管理的下拉按钮
3. 获奖记录页面支持下拉框条件搜索和Excel导出，管理员和教师可以看到所有学生获奖记录
4. 学生可以看到自己的获奖记录，支持条件搜索和Excel导出

## 2. 后端实现

### 2.1 创建获奖记录实体类
- 创建 `StudentAwardRecord.java` 实体类
- 包含字段：id、student_id、award_id、student_name、award_name、award_level、award_type、award_time、create_time、update_time等
- 添加必要的注解和关联关系

### 2.2 创建获奖记录表
- 在 `database_init.sql` 中添加 `student_award_records` 表的创建语句
- 设置合适的字段类型和约束

### 2.3 创建Mapper接口
- 创建 `StudentAwardRecordMapper.java` 接口
- 继承 BaseMapper，实现基本的CRUD操作

### 2.4 创建Service接口和实现类
- 创建 `StudentAwardRecordService.java` 接口
- 创建 `StudentAwardRecordServiceImpl.java` 实现类
- 实现查询、分页查询、条件查询、Excel导出等方法
- 添加权限控制，学生只能查看自己的获奖记录

### 2.5 创建Controller
- 创建 `StudentAwardRecordController.java`
- 添加查询、分页查询、条件查询、Excel导出等接口
- 添加权限控制注解

### 2.6 添加奖项申请通过后的获奖记录生成逻辑
- 在 `StudentAwardApplicationServiceImpl` 的 `adminApproveApplication` 方法中，当管理员审批通过时，自动生成获奖记录

## 3. 前端实现

### 3.1 修改菜单结构
- 修改 `App.vue` 中的菜单配置，将奖项申请和获奖记录合并为奖项管理的下拉菜单
- 使用 `el-sub-menu` 实现下拉菜单
- 奖项管理包含：奖项申请、获奖记录

### 3.2 创建获奖记录视图
- 创建 `StudentAwardRecordView.vue` 组件
- 实现获奖记录的表格展示
- 实现下拉框条件搜索功能（按奖项、学生、获奖时间等）
- 实现Excel导出功能

### 3.3 修改路由
- 在 `router/index.js` 中添加获奖记录路由

## 4. 权限控制

- 管理员和教师可以查看所有学生的获奖记录
- 学生只能查看自己的获奖记录
- 管理员和教师可以导出所有获奖记录
- 学生只能导出自己的获奖记录

- 管理员和教师可以查看所有学生的获奖记录，支持条件搜索和Excel导出
- 学生可以查看自己的获奖记录，支持条件搜索和Excel导出
