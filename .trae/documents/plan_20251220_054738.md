## 问题分析
发送按钮不可见的原因可能是：
1. 布局过于复杂，多个发送按钮分散注意力
2. CSS 样式冲突导致按钮不可见或被隐藏
3. 按钮容器空间不足，导致按钮被截断
4. 按钮样式与背景色对比度不够，难以辨认

## 解决方案
1. **简化按钮布局**：
   - 移除多余的发送按钮，只保留一个明显的发送按钮
   - 将发送按钮放在更明显的位置

2. **修改 CSS 样式**：
   - 确保按钮有足够的对比度，使用明显的颜色
   - 调整按钮的大小和位置，使其更容易找到
   - 移除可能导致按钮不可见的样式

3. **优化布局结构**：
   - 确保输入区域有足够的空间显示发送按钮
   - 调整输入框和按钮的比例，使按钮更突出

## 预期效果
修改后，聊天中心将有一个明显可见的发送按钮，用户可以轻松找到并使用它发送消息。
